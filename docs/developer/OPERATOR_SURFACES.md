# Operator Surfaces

## Implemented now

- REST
- JSON-RPC surface exists
- request/response surface
- templating
- middleware protocol
- OpenAPI JSON
- Swagger UI
- OpenRPC JSON
- Lens / OpenRPC UI
- OAS security scheme modeling for `apiKey`, `http`, `oauth2`, `openIdConnect`, and `mutualTLS`
- auth/security plumbing through `AuthNProvider`, route/app auth configuration, and security dependencies

- WebSockets
- static files
- cookies
- streaming responses
- built-in middleware catalog

## Missing now

- WHATWG SSE
- forms / multipart
- upload handling
- unified `tigrbl` CLI
