# Claim Registry

This is the Phase 11 governance + spec/security + operator-surface + CLI + evidence-lane + Gate B + Gate C claim registry. It records the governance baseline, the core OAS / JSON-RPC / OpenRPC closure from Phase 5, the retained RFC security rows from Phase 6, the Phase 7 docs/operator closure rows, the Phase 8 unified CLI closure rows, the Phase 9 durable evidence-model rows, and the Phase 10 Gate B proof rows.
