# Notes Policy

## Allowed note locations

- work in progress: `docs/notes/wip/YYYY/`
- archived notes: `docs/notes/archive/YYYY/`

## Naming guidance

Use descriptive, date-oriented names such as:

- `YYYY-MM-DD-topic.md`

## Disallowed locations

Do not add:

- root-level scratch notes
- root-level status notes that are not canonical docs
- loose package-level notes that bypass the central docs tree
