# Claim Tiers

| Tier | Meaning | Allowed wording |
|---|---|---|
| Tier 0 | planned / proposed / deferred | planned, proposed, deferred, next target |
| Tier 1 | implemented | implemented, available, experimental |
| Tier 2 | first-party verified | supports, verified in CI |
| Tier 3 | certifiable first-party | certified, conformant, fully compliant, fully featured |
| Tier 4 | independent / external | independently verified, externally validated |

## Rule

Do not use Tier 3+ language for a surface that lacks the evidence required by the current gate model.
